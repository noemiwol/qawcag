Usługi, które oferuję

Testowanie Manualne

Kompleksowe testowanie aplikacji webowych i mobilnych, weryfikacja funkcjonalności oraz ścieżek użytkownika.

Automatyzacja Testów

Automatyzacja testów przy użyciu Selenium, w tym testy regresyjne i E2E.

Audyty Dostępności Cyfrowej

Weryfikacja aplikacji zgodnie z WCAG 2.2, identyfikacja barier i raportowanie zgodności.

Testowanie API

Testowanie RESTful API za pomocą Postmana, analiza przypadków brzegowych i zgodności z dokumentacją.

Mentoring i szkolenia

Oferuję szkolenia z zakresu testowania manualnego, automatyzacji oraz audytów dostępności cyfrowej zgodnie z WCAG 2.2.

Tworzenie scenariuszy testowych

Opracowuję szczegółowe scenariusze i przypadki testowe na podstawie dokumentacji projektowej, dostosowane do specyficznych wymagań aplikacji.
